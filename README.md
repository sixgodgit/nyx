他们都说"越用越懂你"。每家 AI 都这么说。

翻译过来就是：你说了什么，我记下来。下次你再问，我翻出来给你看。这叫"记住"。不叫"懂"。

我们说四件事：

**记住**。每句话加密落沙，一粒不丢。谁也看不见。

**理解**。你不用告诉它你是谁。它从你的沙子里把画像捞出来。你变了，它比你先发现。

**懂你**。不光知道你是谁，还知道你是怎么变成今天这样的。2024 年的决策和现在的决策放在一起，中间那条线就是你的轨迹。

**想你**。三天前你说"等沙漏做完加守夜人"。它还记着。下次启动自己跳出来。不是存数据，是惦记你还没做的事。

---

| | 市面上的记忆系统 | NexSandglass |
|---|---------------|-------------|
| 记住 | 存对话，加标签 | 加密落沙，DPAPI，一粒不丢 |
| 理解 | 静态画像越来越厚 | 偏移超 60% 自动切阶段，旧画像归档成画布 |
| 懂你 | 不知道你变了 | 跨阶段对比，比你自己更早看见变化 |
| 想你 | 关了会话就忘 | 跨会话待办，下次启动自动提醒 |

## 2320 行。零依赖。27KB。

不是砍出来的。是第一性原理推出来的。

消息是字符串，`open("a").write()` 就够了。加密用 DPAPI，一行 import 就够了。搜索用倒排索引，不用向量库。语义有 LLM，不用 embedding。

每加一行先问：**能不写吗？**

## 三层架构

```
写  全平台拦截 → 加密落沙
读  倒排索引 → 解密搜索 → 时间回溯
思  灵魂蒸馏 + 偏移率 + 时间检索 + 织布机
```

| 灵魂蒸馏 | 你是谁 |
| 偏移率 | 你怎么变 |
| 时间检索 | 找什么 |
| 织布机 | 语义搜索 + 矛盾检测 + 跨会话待办 |

## 开始

双击 `install.bat`。

MCP 接入：`{ "mcpServers": { "NexSandglass": { "command": "python", "args": ["mcp_server.py"] } } }`

10 个工具：搜索、最近、时间线、语义搜索、偏移率、综合偏移、搜索滤镜、画像构建、待办查询。
