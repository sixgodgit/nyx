# 偏移率 — 自成长体系

> V1.7.4 | 不是"计算工具"——是沙漏记忆系统的**自成长引擎**。沙子自然落下，偏移自然累积，轮廓自然成形。不判对错，只照见影子往哪边倒。

---

## 什么是偏移率

偏移率 = 主人决策行为的变化方向和幅度。

```
省钱信号正在变实（累计+）  → 偏移率 +55% frugal
花钱轮廓正在成形（累计-）  → 偏移率 -40% spend
放弃倾向的影子（累计--）  → 偏移率 -80% drift
```

不是"主人偏了 55% 扣分"，是"影子往省钱方向倒了 55%——轮廓正在成形"。

---

## 偏移率怎么算

### 2D 本地模式（80 分）

1. **决策粒子落盘** — pulse.py 每收到决策，`_detect_chain()` 拆链条，`_direction()` 判方向
2. **加权平均** — `comprehensive_offset()` 读最近 50 条粒子，加权平均，越近权重越大
3. **波浪自吸收** — EMA 衰减 + β 回归基线，连续同向 → 合并成波
4. **镜子敏感度** — `_WAVE_THRESHOLDS` 控制"多少层影子算成形"

### 3D LLM 模式（+120 分）

- `_synthesize_3d()` — LLM 吃画像+粒子+偏移+织布机 → 合成立体像
- LLM 自己决定提醒语气（好奇式/陪伴式/安静式）
- 结果写入 `3d_annotations.jsonl` 永久保存

## 偏移率的用途

| 用途 | 怎么用 |
|------|--------|
| **画像生长** | `persona_update()` 末尾注入偏移率摘要——画像随偏移率持续生长 |
| **搜索偏置** | `search_filter()` 四维权重——粒子标签 → 搜索偏好 |
| **阶段切换** | 连续 2 次偏移超 60% → 静默切阶段 |
| **织布机矛盾** | 画像说 A 但偏移率说 B → 矛盾检测 |
| **伴侣提醒** | `entropy_reminder()` 情绪熵驱动——高熵安静/低熵热情 |

## 偏移率的配置

```python
# sandglass_think.py 第 648 行
_WAVE_THRESHOLDS = {
    "frugal": {"contour": 50},        # 省钱影子叠 50 层→成形
    "spend":  {"contour": 50},        # 同上
    "drift":  {"contour": 30,         # 放弃更敏感（30 层就注意）
               "放弃": 100, "妥协": 60, "烦躁": 30},  # 三档权重
}
```
