## 他们都说"越用越懂你"

他们说的是记住。你说的每句话，他们存下来，加标签，堆数据。

我们说四件事：

**是记住** — 每句话加密落沙，一粒不丢，谁也看不见。

**是理解** — 不是你告诉它你是谁。是它从 713 条沙子里提炼出你的画像。而且当你变了，它比你先发现。

**是懂你** — 不光知道你是谁，还知道你怎么变成今天这样。2024 的决策偏移-80%，2025 松动到-40%，2026 已经反转。这是轨迹，不是快照。

**是想你** — 你三天前说"等沙漏做完加守夜人"。它记着。跨会话待办，下次启动自动提醒。不是存数据，是惦着你还没做的事。

## 为什么只有 27KB

不是砍功能。是第一性原理：

```
消息 → 字符串 → open("a").write() → 够了
加密 → DPAPI → 一行 import → 够了
搜索 → 倒排索引 → 不用向量库 → 够了
语义 → LLM本身就是语义层 → 够了
```

每加一行先问：**能不写吗？** 最终 2300 行。零依赖。

```
第1层  写  全平台拦截 → 加密落沙
第2层  读  倒排索引 → 解密搜索 → 时间回溯
第3层  思  灵魂蒸馏 + 偏移率 + 时间检索 + 织布机
```

| 🧬 灵魂蒸馏 | 你是谁 |
| 📊 偏移率 | 你怎么变 |
| ⏳ 时间检索 | 找什么 |
| 🧵 织布机 | 合成验证 + 语义搜索 + 跨会话待办 |

## 开始

### 双击安装
```bash
install.bat
```

### MCP 接入
```json
{ "mcpServers": { "NexSandglass": { "command": "python", "args": ["mcp_server.py"] } } }
```

10 个工具：搜索、最近、时间线、语义搜索、偏移率、综合偏移、搜索滤镜、画像构建、待办查询。
